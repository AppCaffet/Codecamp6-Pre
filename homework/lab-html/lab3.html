Lab 1.3 ให้หา input type มีอะไรบ้าง

Tag description
text define a single line text input
password define a password field
submit define a button for submitting form handler
radio define radio button. let user select ONLY ONE of a limited number of choice
checkbox define a checkbox. let user select zero or more option of limited number of choices
button define button
color is used for input fields that showed contain a color
date is used for input fields that should contain a date
datetime-local specifies a date and time input field with no time zone
email is used for input fields that should contain an email address
file define a file select and browse button to file uploads
number define numberic input field
