Lab 1.1 - Attribute

Tag Attributes Description
<h> style ใช้ความสามารถของ css ในการจัดการตกแต่งส่วนที่เป็น "หัวข้อ"
<p> style ใช้ความสามารถของ css ในการจัดการตกแต่งส่วนที่เป็น "ข้อความ"
<div> class,id ใช้เพื่อระบุในการเข้าไปจัดการกับ tag div นั้นๆ
<img> alt ใช้ขึ้นข้อความรูปเวลาไม่ขึ้น
height ใช้ระบุความสูงของรูป
sizes ใช้ระบุขนาดสำหรับแต่ละขนาดหน้าจอ
src ใช้ระบุไปยังตำแหน่งของรูปภาพ
width ใช้ระบุความกว้างของรูปภาพ
<a> download ใช้สำหรับการระบุให้ user ดาวน์โหลด
href ใช้ระบุอ้างอิงไปยัง URL ที่ต้องการ
target ระบุตำแหน่งการเปิดหน้าต่างเวลา user กด
<ul> onclick ใช้ทำงานเวลาเม้าส์มีการคลิกเข้ามา
ondblclick ใช้ทำงานเวลาเม้าส์มีการดับเบิ้ลคลิกเข้ามา
ondrag ใช้ในการลากวาง
onscroll ใช้ในทำเป็น scroll bar
<ol> reversed ใช้เรียงเลขจากมากไปน้อย
start ใช้ระบุตำแหน่งตัวเริ่มได้
type ใช้กำหนดได้ว่าจะให้แสดงผลอออกมาเป็นชนิดไหนใน order list
<form> action ใช้ระบุว่าหลังจากกด submittedจะให้ข้อมูลไปทำอะไรต่อ
autocomplete ช่วยจดจำข้อมูลที่เคยใส่ input เข้าไป สามารถให้ on/off ได้
method ใช้ระบุให้ http รู้ว่าจะส่งค่าข้อมูลไปแบบไหน get/post
name เป็นการใส่ชื่อให้กับฟอ์มของเรา
target ใช้แสดงหน้าต่างเวลา user กด submitted
<table> style ช้ความสามารถของ css ในการจัดการตกแต่งส่วนที่เป็น "ตาราง"
<button> autofocus ใช้ระบุความโฟกัสตอนดปุ่ม
disabled ใช้ระบุว่าปุ่มนี้กดไม่ได้
name ตั้งชื่อให้กับปุ่ม
type ใช้ระบุชนิดปุ่ม
value ใช้ระบุค่าให้กับปุ่ม
