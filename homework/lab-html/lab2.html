Lab 1.2 ให้หาความเหมือนและความแตกต่างระหว่าง 2 tags ที่กำหนดให้

Tag สิ่งที่เหมือน สิ่งที่แตกต่าง
<hr> & <br> Empty HTML Element <br> - tags defines a line break
<hr> - ขีดเส้นแบ่งหน้า
<ul> & <ul> Nested HTML Element <ul> - unorder list
<ul> - order list
<select> & <checkbox> & <radio> Nested HTML Element <select> - ใช้ใน dropdown list
<checkbox> - เป็น input type สามารถเลือกได้ตั้งแต่ zero - more
<radio> - เป็น input typ สามารถเลือกได้แค่อย่างเดียว
<div> & <span> Nested HTML Element <div> - เป็น display แบบ block
<span> - เป็น display แบบ inline
<script> & <style> Nested HTML Element & Element in the tag head <script> - for javascript
<style> - for css
